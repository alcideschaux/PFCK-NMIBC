CD8-to-FOXP3 regression

# Loading the datasets
load("../R/df.Rdata")
# Sourcing scripts
scripts <- list.files(
    path = "~/Dropbox/R-scripts",
    pattern = "*.R$",
    full.names = TRUE
)
for (script in scripts) { source(script) }
# Global options
opts_chunk$set(message = FALSE, warning = FALSE)
# Regression models
df_pairs <- df_pairs %>% 
    mutate(
        cd8foxp3_tumor = ifelse(
            cd8_tumor_count_max > foxp3_lymph_tumor_max,
            "CD8 > FOXP3", "CD8 <= FOXP3"
        ),
        cd8foxp3_stroma = ifelse(
            cd8_stroma_count_max > foxp3_lymph_stroma_max,
            "CD8 > FOXP3", "CD8 <= FOXP3"
        )
    )
predictors <- with(df_pairs, list(
    "CD8 > FOXP3 in lymphocytes infiltrating tumor" = cd8foxp3_tumor,
    "CD8 > FOXP3 in lymphocytes infiltrating stroma" = cd8foxp3_stroma
))
varlabels <- names(predictors)

METHODOLOGY: Odds ratios were estimated using binary logistic regression. A 2-level category of the CD8-to-FOXP3 balance was built comparing the counts of positive lymphocytes in tumor and in stroma, as follows: CD8 counts greater than FOXP3 counts vs.Â CD8 counts less than or equal to FOXP3 counts.
